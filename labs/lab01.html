Lab 1: Setting Up Your Environment

Install Visual Studio Code, add the GitHub Copilot extension, sign in with your GitHub account, and verify everything is working with your first AI-powered code suggestion.

🎯Objectives

  • Install Visual Studio Code on your machine
  • Install the GitHub Copilot and GitHub Copilot Chat extensions
  • Authenticate with your GitHub account
  • Verify Copilot is active and providing suggestions
  • Accept your first Copilot suggestion

📋Prerequisites

  • A GitHub account (free accounts get Copilot Free tier)
  • A computer running Windows, macOS, or Linux
  • Internet connection

📝Step-by-Step Instructions

1Install Visual Studio Code
  1. Open your browser and navigate to https://code.visualstudio.com
  2. Click the large "Download" button — it will detect your operating system automatically
  3. Run the installer:
    • Windows: Run the downloaded .exe file. Check "Add to PATH" during installation.
    • macOS: Open the .dmg file and drag VS Code to your Applications folder.
    • Linux: Install the .deb or .rpm package, or use snap: sudo snap install code --classic
  4. Launch VS Code to verify the installation
2Install the GitHub Copilot Extension
  1. In VS Code, click the Extensions icon in the left sidebar (or press Ctrl+Shift+X)
  2. Search for "GitHub Copilot"
  3. Click Install on the extension by GitHub (it has a verified badge ✓)
  4. This will also install GitHub Copilot Chat automatically as a dependency
  5. Wait for the installation to complete — you'll see a notification in the bottom-right corner
💡
As of 2025, the Copilot and Copilot Chat extensions have been merged. Installing "GitHub Copilot" gives you both inline suggestions and the chat panel.
3Sign In to GitHub
  1. After installing the extension, you'll see a "Sign in to GitHub" prompt in the bottom-right corner
  2. Click the prompt (or click the Copilot icon in the status bar at the bottom)
  3. A browser window will open asking you to authorize VS Code
  4. Sign in with your GitHub credentials
  5. Click "Authorize Visual-Studio-Code"
  6. Return to VS Code — you should see a confirmation that you're signed in
ℹ️
If you have a free GitHub account, you'll be on the Copilot Free tier with 2,000 completions and 50 chat messages per month. This is enough for this entire course!
4Verify Copilot is Active
  1. Look at the status bar at the bottom of VS Code
  2. You should see a Copilot icon (looks like a small two-toned sparkle or pilot icon)
  3. If the icon is present and not showing a crossed-out symbol, Copilot is active
  4. Click the icon to see status details — it should say "Copilot is ready"
✅ Expected Result
The Copilot icon appears in the status bar without any error indicators. Clicking it shows "GitHub Copilot is ready" or similar active status message.
5Accept Your First Suggestion
  1. Create a new file: File → New File, then save it as hello.py
  2. Type the following comment on line 1:
hello.py 
# A function that greets a user by name and returns the greeting message
  1. Press Enter to go to a new line
  2. Start typing: def greet
  3. Wait a moment — you'll see ghost text appear in a dimmed color suggesting the rest of the function
  4. Press Tab to accept the suggestion
✅ Expected Output
Copilot should suggest something like:
hello.py — Expected Suggestion 
# A function that greets a user by name and returns the greeting message
def greet(name: str) -> str:
    return f"Hello, {name}! Welcome aboard."

# Test the function
print(greet("Alice"))
💡
Tip: Your suggestion may look slightly different — that's normal! Copilot generates suggestions probabilistically, so the exact wording may vary. The important thing is that it correctly generated a function body from your comment.
6Try Copilot Chat
  1. Open Copilot Chat by clicking the chat icon in the sidebar (or press Ctrl+Shift+I)
  2. Type: "Explain what this file does"
  3. Copilot Chat will analyze your hello.py file and provide a natural language explanation
  4. Try another prompt: "Add error handling to the greet function for empty names"
  5. Review the suggested code changes and click "Apply" to insert them
✅ Expected Result
Copilot Chat responds with a clear explanation and provides updated code with input validation, such as checking for empty strings or None values.

💡Tips & Best Practices

💡
  • • Keyboard shortcuts to remember: Tab (accept), Esc (dismiss), Alt+] / Alt+[ (cycle suggestions), Ctrl+Enter (see all suggestions)
  • • Copilot works best with descriptive comments and clear function names
  • • Save your files with the correct extension (.py, .js, .ts) so Copilot knows the language
  • • If suggestions don't appear, check the status bar icon and ensure you're connected to the internet
🏆 Extra Credit Challenge

Create a new file called fibonacci.py. Write only a comment describing a Fibonacci function, then let Copilot generate the entire implementation. Try getting Copilot to generate both a recursive and an iterative version by changing your comment.
