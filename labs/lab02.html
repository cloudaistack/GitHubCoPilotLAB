Lab 2: Hello Copilot — Build a Calculator

Experience the magic of AI-powered coding. You'll write only function signatures and comments — Copilot will generate all the implementation code for a fully working calculator application in Python.

🎯Objectives

  • Use comment-driven development to guide Copilot
  • Write function signatures and let Copilot complete the bodies
  • Cycle through multiple suggestions using keyboard shortcuts
  • Build a complete, working Python calculator

📋Prerequisites

  • Completed Lab 1 (VS Code + Copilot installed and working)
  • Python 3.8+ installed on your system
  • Basic Python knowledge (variables, functions, input/output)

📝Step-by-Step Instructions

1Create the Project File
  1. Create a new folder called copilot-calculator
  2. Open it in VS Code: File → Open Folder
  3. Create a new file called calculator.py
  4. Start by typing the module docstring below — this sets the context for Copilot:
calculator.py — Type this 
"""
A command-line calculator application that supports basic arithmetic
operations: addition, subtraction, multiplication, division, and modulo.
Includes input validation and error handling.
"""
2Generate Arithmetic Functions

Type each function signature below, one at a time. After typing the signature, press Enter and wait for Copilot's ghost text suggestion, then press Tab to accept.

calculator.py — Type these signatures one by one 
def add(a: float, b: float) -> float:

def subtract(a: float, b: float) -> float:

def multiply(a: float, b: float) -> float:

def divide(a: float, b: float) -> float:

def modulo(a: float, b: float) -> float:
💡
Key Insight: Notice how after writing the first 1-2 functions, Copilot starts predicting the pattern for subsequent functions more accurately. This is few-shot prompting in action — you established a pattern and Copilot follows it.
✅ Expected Output
Copilot should generate simple return statements for each function. The divide function should ideally include a check for division by zero.
calculator.py — Expected Copilot output 
def add(a: float, b: float) -> float:
    """Return the sum of a and b."""
    return a + b


def subtract(a: float, b: float) -> float:
    """Return the difference of a and b."""
    return a - b


def multiply(a: float, b: float) -> float:
    """Return the product of a and b."""
    return a * b


def divide(a: float, b: float) -> float:
    """Return the quotient of a divided by b. Raises ValueError if b is zero."""
    if b == 0:
        raise ValueError("Cannot divide by zero")
    return a / b


def modulo(a: float, b: float) -> float:
    """Return the remainder of a divided by b."""
    if b == 0:
        raise ValueError("Cannot modulo by zero")
    return a % b

💡Tips & Best Practices

💡
  • • Descriptive names matter: get_number() produces better suggestions than gn()
  • • Type hints help: Adding -> float tells Copilot what to return
  • • Cycle suggestions: Press Alt+] to see alternatives if the first suggestion isn't ideal
  • • Comment-first: Write a detailed comment, then the function signature — Copilot combines both for context
